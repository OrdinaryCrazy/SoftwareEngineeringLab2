\subsection{R.INTF.CALC.001: 一对一即时通讯功能}
这是聊天软件最基本的功能。
\subsubsection{介绍}
对用户而言，该功能的基本需求为:
\begin{itemize}
  \item 用户可以向好友发送聊天信息。
  \item 用户可以接收到好友发送的聊天信息。
\end{itemize}
\subsubsection{输入}
向好友发送的聊天信息。
\subsubsection{处理}
\begin{itemize}
  \item 向服务器发送聊天信息。
  \item 如果好友在线，则服务器立即转发该信息；否则，进行缓存，直到好友上线。
  \item 使用一个监听线程监听服务器，接收服务器转发的信息。
\end{itemize}
\subsubsection{输出}
展示接收到的聊天信息。
